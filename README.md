# Hello I am Keno! 👋

I am an Electronics Engineering student from PUP Manila. I love to work on computer vision and AI/ML integrations into it. As of recent, I've been particularly interested and have been doing research on quantum image processing.

**My favorite projects right now:**

- [**open-quantum-computer-vision**](https://github.com/Keno-00/open-quantum-computer-vision) - OpenCV-equivalent for quantum image processing
- [**MHRQI**](https://github.com/Keno-00/MHRQI) - A multiscale quantum image representation that acts as a recursive subdivision. I might try to do improvements to this
- [**QARQI**](https://github.com/Keno-00/QARQI) - A fixed 5 qudits quantum image representation that uses assumed infinite levels
